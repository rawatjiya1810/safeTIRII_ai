# 🛡️ SafeTIRRI AI

> Protecting Every E-Rickshaw. Protecting Every Livelihood.

SafeTIRRI AI is an AI-powered cybersecurity platform designed to secure connected electric rickshaws against emerging digital threats. As India's EV ecosystem rapidly expands, connected vehicles become increasingly vulnerable to cyber risks. SafeTIRRI empowers e-rickshaw drivers with AI-driven cybersecurity tools, digital health monitoring, cyber awareness, and emergency assistance—creating a safer and smarter transportation ecosystem.

---

## 🚀 Overview

SafeTIRRI AI is built to bridge the cybersecurity gap in India's connected electric mobility ecosystem.

The platform enables drivers to assess their vehicle's cyber health, understand security risks through an AI-powered coach, access emergency assistance, and report cyber incidents to relevant authorities.

Designed as a production-ready MVP, SafeTIRRI demonstrates how AI can strengthen cybersecurity awareness and resilience for connected electric vehicles.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/your-username/SafeTIRRI-AI.git
```

Go into the project directory

```bash
cd SafeTIRRI-AI
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Backend

```bash
pip install -r requirements.txt

uvicorn main:app --reload
```
